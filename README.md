# بوت تليجرام للجدولة التلقائية

بوت تليجرام يعمل على GitHub Actions مجاناً لجدولة الرسائل كل 15 دقيقة.

## المميزات

- ✅ جدولة 96 رسالة يومياً
- ✅ فاصل 15 دقيقة بين الرسائل
- ✅ يعمل 24/7 على GitHub مجاناً
- ✅ توقيت ليبيا
- ✅ تقسيم وخلط الرسائل

## الأوامر

- `جدول` - جدولة كاملة لليوم
- `جدولة اليوم` - جدولة لبقية اليوم
- `جدولة 24 ساعة` - جدولة لـ24 ساعة
- `تقسيم` - تقسيم الرسائل
- `فحص` - فحص حالة البوت
- `حذف المجدول` - حذف الرسائل المجدولة
- `مساعدة` - عرض جميع الأوامر

## الإعداد

1. انسخ هذا المستودع
2. اضغط على Settings → Secrets → Actions
3. أضف الأسرار التالية:
   - `API_ID`: رقم API ID
   - `API_HASH`: رمز API HASH
   - `SESSION_STRING`: جلسة التليجرام

4. اذهب إلى Actions → Telegram Bot Runner → Run workflow
